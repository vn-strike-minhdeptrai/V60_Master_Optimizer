# V60_Master_Optimizer v2.0

**Dynamic Profile Optimizer cho LG V60 ThinQ (APatch)**

Gửi trào lưu giữ tinh túy LG: Quad DAC, LG Music, Camera + tối ưu hiệu năng, nhiệt độ, pin.

## Tính năng chính
- 3 Profile động: **Game** / **Balanced** / **Battery Saver**
- WebUI Anime style đẹp (glassmorphism, Vietnamese)
- Safe Debloat (whitelist editable)
- Sysfs tweaks (CPU/GPU governors, thermal)
- Logging chi tiết
- Reversible, an toàn

## Cài đặt (Termux)
1. git clone https://github.com/vn-strike-minhdeptrai/V60_Master_Optimizer.git
2. cd V60_Master_Optimizer
3. sh install.sh   # or manual copy to /data/adb/modules/

## Sử dụng WebUI
Mở WebUI qua APatch app hoặc Termux.
